Weather vidget

  1. На данной странице реализован виджет погоды
  2. Он показывает текущую погоду используя данные, полученные в результате запроса на публичную API
  3. На текущую страницу данный виджет добавляется с помощью команды new WeatherVidget().getWeather(); Через сторонний файл.
  4. То есть этот виджет можно добавить при необходимости на любую страницу, подключив WeatherVidget.js
  5. Также есть кнопка - погода на три дня. По клику на которую отправляется второй запрос и обрабатывает данные погоды на три дня
  6. При необходимости данный виджет можно свернуть или показать
